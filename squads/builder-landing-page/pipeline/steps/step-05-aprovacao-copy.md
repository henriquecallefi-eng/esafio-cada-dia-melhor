# Step 05 — Aprovação do Copy

**Type:** Checkpoint  
**Agent:** User  
**Input:** `copy-deck.md` (Carlos)  
**Output:** Copy selection (proceed to Diana Design)

## Descrição

User revisa as variantes de copy de Carlos e escolhe qual headline, subheadline, proof e CTA usar.

## Checklist

- [ ] Headlines são claros e param o scroll?
- [ ] Variantes são genuinamente diferentes (benefit vs. problem vs. transformation)?
- [ ] Subheadline apoia o headline?
- [ ] CTA é acionável (não vago)?
- [ ] Nenhuma frase usa palavras proibidas ("incrível", "fantástico")?

## Se Aprovar

Indique qual variante de headline, subheadline, CTA prefere.

## If Adjust

Descreva feedback. Carlos executará novamente.
