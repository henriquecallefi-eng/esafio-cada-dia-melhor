# Step 07 — Aprovação do Design System

**Type:** Checkpoint  
**Agent:** User  
**Input:** `design-system.md` (Diana)  
**Output:** Approval (proceed to Henrique HTML)

## Descrição

User revisa o design system — paleta, tipografia, espaçamento, componentes.

## Checklist [Modo B]

- [ ] Cores extraídas correspondem ao design Stitch?
- [ ] Tipografia está clara (H1-H3, body, small)?
- [ ] Espaçamento segue escala 8px?
- [ ] Componentes (buttons, cards, etc) estão definidos?

## Checklist [Modo A]

- [ ] Paleta alinha com o positioning (tech = azul, wellness = verde)?
- [ ] Tipografia tem hierarquia clara?
- [ ] Espaçamento é consistente (base 8)?
