# Princípios de Conversão para Landing Pages

## 1. Estrutura AIDA

- **Attention:** Hook que para o scroll (0-3 segundos)
- **Interest:** Promise expandida (o que ganha se ficar)
- **Desire:** Prova social, cases, testimoniais
- **Action:** CTA clara e simples

## 2. Above the Fold

Tudo que importa deve estar visível sem scroll:
- Headline
- Subheadline
- CTA principal
- 1 imagem de suporte

## 3. Tipografia para Conversão

- **Headline:** 48px+, peso 700, legível de longe
- **Subheadline:** 20-24px, peso 400-600, apoia o headline
- **Body:** 16px+, linha-height 1.6+, fácil de ler
- **CTA:** 16px+, peso 600, contraste excelente

## 4. Cores para Conversão

- **CTA button:** Contraste máximo (ex: azul em branco)
- **Trust colors:** Azul (tech), verde (wellness), preto (luxury)
- **Evite:** Vermelho puro (agressivo), rosa (tendência passada)

## 5. CTA Design

- **Mínimo 44px height** (toca fácil em mobile)
- **Padding: 12px 24px+** (não aperta)
- **Borda-radius: 4-8px** (moderno, não demais)
- **Sem mais de 2 cores** no button (confunde)

## 6. Prova Social

- **Quantitativa:** "10.000+ clientes satisfeitos"
- **Qualitativa:** 1-3 depoimentos com foto + nome + empresa
- **Logos:** 3-5 logos de clientes conhecidos

## 7. Mobile Primeiro

- Single column layout
- Menus colapsáveis (hamburger)
- Buttons 48px+ de altura
- Imagens otimizadas (< 200KB cada)

## 8. Urgência (Opcional)

- "Apenas 3 vagas por mês" (scarcity real)
- "Desconto 40% para primeiros 100" (deadline real)
- NÃO: "Oferta limitada" (sem dizer quando acaba)
