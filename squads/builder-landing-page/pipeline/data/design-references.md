# Referências de Design — Paletas e Tipografias Recomendadas

## Tech Startups
- **Cores:** Azul (#2563eb), Cinza (#404040), Branco (#FFFFFF)
- **Font:** Inter, Poppins, SF Pro Display
- **Exemplo:** Stripe, Slack, GitHub

## Wellness/Health
- **Cores:** Verde (#10B981), Branco, Cinza-claro
- **Font:** Lora, Georgia, Playfair Display
- **Exemplo:** Headspace, Calm

## E-commerce
- **Cores:** Preto (#1A1A1A), Branco, Accent (vermelho ou rosa)
- **Font:** Montserrat, Playfair Display
- **Exemplo:** Shopify, Etsy

## Education
- **Cores:** Azul, Laranja, Branco
- **Font:** Rubik, Open Sans, Lora
- **Exemplo:** Udemy, Coursera

## Community/Social
- **Cores:** Roxo (#8B5CF6), Rosa, Branco
- **Font:** Inter, Poppins, Space Mono
- **Exemplo:** Discord, Notion

## Dicas Gerais

- Use no máximo **2-3 cores principais**
- Sempre inclua Neutral scale (branco → preto)
- Escolha tipografia com **pelo menos 2 pesos** (regular + bold)
- Considere **acessibilidade** (contraste >= 4.5:1)
