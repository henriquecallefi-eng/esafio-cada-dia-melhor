# Anti-Patterns a Evitar

## Copy
1. **Adjetivos vazios** — "Incrível", "Fantástico", "Melhor"
2. **Headlines genéricas** — "Bem-vindo", "Saiba mais"
3. **Frases muito longas** — Max 12 palavras no headline
4. **CTA vago** — "Clique aqui" em vez de "Agendar visita"

## Design
1. **Muitas cores** — Limit a 3-4 principais + neutrals
2. **Fonte pequenininha** — Min 14px body, 48px headline
3. **Imagens genéricas** — Stock photos óbvias (iStock watermark visible)
4. **Contraste ruim** — Cinza-claro em branco é invisible

## Layout
1. **Muita informação acima do fold** — Hero + CTA só
2. **Sem hierarquia** — Todos os elementos parecem igualmente importantes
3. **Mobile ignorado** — Desktop-only designs
4. **Formulários longos** — Max 3 campos para lead, 5 para purchase

## Performance
1. **Imagens não otimizadas** — 1MB+ por imagem
2. **Muitas fonts** — Max 2 font families
3. **CSS/JS não minificado** — Arquivo > 500KB
4. **Sem lazy-load** — Imagens below-fold carregam de uma vez

## Animações
1. **Auto-play video com som** — Irritante
2. **Muitas animações** — Parecem amateur
3. **Animações lentas** — > 1s é slow
4. **Sem prefers-reduced-motion** — Acessibilidade falha

## SEO
1. **Sem title tag** — ou genérico ("Home")
2. **Sem meta description**
3. **H1 repetido**
4. **Imagens sem alt text**
