# Arquitetura de Conversão — Landing Pages Vencedoras

## 🎯 Filosofia

Uma landing page vencedora não é bonita — é **estratégica**. Cada elemento, cada palavra, cada clique serve a um propósito: **converter visitante em lead/cliente**.

---

## 📐 Estrutura Modular Obrigatória (Hero → Value → Social Proof → CTA → FAQ → Final CTA)

### 1️⃣ Hero Section (Above the Fold — 0-100vh)

**Objetivo:** Capture atenção em 3 segundos. Responda: "Quem você é?" e "Por que devo ficar?"

**Elementos obrigatórios:**
- **Headline principal** (H1)
  - Deve incluir: Benefício principal OU problema que resolve
  - Comprimento: 8–12 palavras (máx)
  - Linguagem: Imperativa ou transformacional, não descritiva
  - ❌ "Software de gestão de projetos"
  - ✅ "Entregue projetos 3x mais rápido sem chaos de emails"

- **Subheadline** (suporta, detalha)
  - Diferencial ou prova rápida
  - "Usado por 500+ equipes, $10M em tarefas gerenciadas"

- **CTA Primary** (botão destacado)
  - Cor: Contraste mínimo 4.5:1 com fundo (WCAG AA)
  - Texto de ação: "Iniciar Grátis" / "Ver Demo" (específico)
  - Hover state: Visual feedback claro (cor, sombra, escala)

- **Visual (Hero Image / Video / Mockup)**
  - Mockup do produto OU vídeo de 10-15s
  - NUNCA: Stock photos genéricos (reduz credibilidade)
  - Otimizado: <200KB para mobile

- **Background**
  - Gradiente suave OU cor sólida (não distraia da copy)
  - Contraste com texto >= 4.5:1

---

### 2️⃣ Value Proposition Section (100-250vh)

**Objetivo:** Esclareça POR QUÊ eu deveria escolher você (não um concorrente).

**Padrão: 3 Blocos de Valor**

Cada bloco:
```
[ÍCONE]
Benefício (7-10 palavras)
Detalhe em 1 frase
```

**Exemplo forte (SaaS):**
```
💰 Economize 20h/semana
Automação elimina tarefas manuais repetitivas

🚀 Entregue 3x mais rápido
Workflows inteligentes aceleram ciclos

🔒 Segurança enterprise
SOC2 Type II, criptografia end-to-end
```

**Exemplo fraco (evitar):**
```
✨ Incrível
Extremamente bom para seu negócio

🌟 Revolucionário
Muda tudo como você pensa

⭐ Melhor em Classe
Todos adoram este produto
```

**Anti-patterns:**
- Nenhum número? Falta credibilidade.
- Mais de 5 blocos? Paralisa decisão.
- Texto genérico ("Ótimo", "Incrível")? Baixa confiança.

---

### 3️⃣ Social Proof Section (250-400vh)

**Objetivo:** Reduza fricção psicológica ("É seguro confiar?").

**Elementos obrigatórios:**

1. **Logos de clientes** (5-10)
   - Maior = melhor (confiança)
   - Real > fictional

2. **Depoimentos com rosto + nome + empresa**
   ```
   "Economizamos $200k em 6 meses. Impecável."
   — Maria Silva, CTO, TechCorp Brasil
   ⭐⭐⭐⭐⭐
   ```
   - Foto real (não AI gerada — cria desconfiança)
   - Número específico (não "muito", "bastante")
   - Star rating visual

3. **Métricas públicas** (se houver)
   - "500+ equipes ativas"
   - "10M+ tarefas gerenciadas"
   - "$50M+ economizados"
   - ⏰ Data atualizada (2024, não "mais de 1000")

4. **Case study (resumido)**
   ```
   Resultado: "Aumentamos conversão de 2% para 5% em 3 meses"
   Como: "Implementamos fluxo de automação customizado + análise de comportamento"
   Impacto: "+150% em leads qualificados"
   ```

### 4️⃣ Gatilhos Mentais Obrigatórios

Integrar ao longo da página (não tudo em uma seção):

| Gatilho | Onde | Exemplo |
|---------|------|---------|
| **Escassez** | CTA ou seção de preço | "Últimas 50 contas com 50% desconto" |
| **Urgência** | Hero ou rodapé | "Oferta válida até 31/dez" |
| **Autoridade** | Social proof | Logos de marcas conhecidas, certificações |
| **Aprovação Social** | Depoimentos | "1000+ reviews ⭐⭐⭐⭐⭐" |
| **Reciprocidade** | Lead magnet | "Baixe guia grátis de 50 páginas" |
| **Curiosidade** | Subheadlines | "3 razões inesperadas por que X é melhor que Y" |

**Anti-patterns:**
- Escassez falsa ("Apenas 3 vagas!" — todo dia) → Morte de trust
- Urgência exagerada → Spam
- Autoridade fabricada (logos de sites aleatórios) → Desconfiança

---

### 5️⃣ FAQ / Objeções (400-550vh)

**Objetivo:** Responda as 5-7 perguntas que matam a conversão.

**Padrão: Acordeão (Accordion)**

```
❓ Quanto custa?
[Expandir] "Começamos em R$ 99/mês. Sem contrato. Cancele quando quiser."

❓ Preciso de cartão de crédito para trial?
[Expandir] "Não. Trial de 14 dias gratuito, sem cartão."

❓ Quanto tempo para onboarding?
[Expandir] "15 minutos de setup + demo guiada. Média: 2 dias até usar em produção."
```

**Estrutura obrigatória:**
- Pergunta é um pain point real (dados do Beatriz Briefing)
- Resposta é 1-2 frases (concisa)
- Se precisa de mais detalhe → link para doc externa

---

### 6️⃣ CTA Final (550-600vh)

**Objetivo:** Conversão final. Reforce o mesmo CTA do hero.

**Padrão:**

```

║                                                                  ║
║  Últimas dúvidas?                                                ║
║                                                                  ║
║  [Botão Principal: Iniciar Grátis]  [Link: Agendar Demo]        ║
║                                                                  ║
║  Regressão de dúvida?                                            ║
║  Email: suporte@empresa.com | Chat: (disponível 9-18 BRT)       ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

**Cores:**
- Botão primário: Contraste 4.5:1 mínimo, cor "CTA vibrante" (varia por indústria)
  - SaaS: Azul brilhante, Verde, Roxo
  - E-commerce: Laranja, Vermelho
  - Educação: Azul, Verde
  - B2B: Azul escuro, Preto

## 🎨 Padrão de Cores para Conversão

### Tech/SaaS
- **CTA:** Azul vibrante (#0066FF, #5B21B6)
- **Acento:** Roxo ou Ciano
- **Background:** Branco, Cinza claro
- **Texto:** Preto/Cinza escuro
- **Efeito:** Sombra suave em card, hover lift

### E-commerce
- **CTA:** Laranja ou Vermelho (#FF6B35, #DC2626)
- **Acento:** Amarelo (spotlight)
- **Background:** Branco limpo
- **Texto:** Preto
- **Efeito:** Animação pulse no CTA

### Wellness/Lifestyle
- **CTA:** Verde (#22C55E) ou Teal (#14B8A6)
- **Acento:** Rose/Coral
- **Background:** Branco, Off-white
- **Texto:** Cinza escuro
- **Efeito:** Suavidade, transições lentas

## ✍️ Copy Patterns Vencedores

### Headline Pattern #1: Benefício + Diferencial

**Fórmula:**
```
[Verbo de ação/transformação] [Resultado mensurável] [Diferencial único]

✅ "Entregue projetos 40% mais rápido sem abandonar sua stack favorita"
✅ "Aumente conversão 3x com uma única landing page (sem código)"
✅ "Reduza bug reports em 50% com testes automáticos que você escreve em português"
```

### Headline Pattern #2: Problem-Agitate-Solve

```
[Problema] é [Resultado ruim]. 

[Agitação - Por quê é ruim]?

[Solução].

✅ "Planilhas são lentas. Seu time gasta 10h/semana digitando dados. 
Nós automatizamos — zero cliques, data real-time."
```

### Subheadline Pattern: Prova + Diferencial

```
[Número impressive] [Métrica] em [Prazo]. [Diferencial].

✅ "500+ equipes produzem 50% mais em 30% menos tempo. Sem treinamento."
✅ "$200M economizados globalmente. Sem integração complexa."
```

## 📊 Checklist de Conversão Antes de Deploy

- [ ] Hero H1 contém número OU transformação específica (não genérico)
- [ ] Cada bloco de valor tem métrica (não adjetivo)
- [ ] Social proof tem 5+ logos reais + 3+ depoimentos com foto/nome
- [ ] CTA primário tem contraste >= 4.5:1 WCAG AA
- [ ] FAQ responde as 5-7 objeções reais (data do Beatriz)
- [ ] Falta urgência/escassez? (sem forçar, sem mentir)
- [ ] Copy é ação-orientado (verbos imperativos: Comece, Descubra, Aumente)
- [ ] Cada seção tem ONE objetivo claro (não 3)
- [ ] Mobile: Texto legível, CTA acessível, imagens otimizadas (<200KB)
- [ ] Nenhuma jargon técnica que público-alvo não entende

## 🔄 Próximas Iterações (A/B Testing)

**Após primeira versão:**

1. **Headlines:** Teste 3 variantes em GA4 por 2 semanas
2. **CTA Color:** Vermelho vs Azul vs Verde → qual converte mais?
3. **Prova Social:** Logos + depoimentos vs Apenas depoimentos
4. **Form Length:** 3 campos vs 1 campo (email) vs Só botão
5. **FAQ:** Acordeão aberto vs fechado na entrada

**Métrica de sucesso:** Conversion Rate >= 3% (SaaS) / >= 2% (B2B) / >= 5% (E-commerce)
